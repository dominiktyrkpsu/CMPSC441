# Prompt Engineering Process
The goal of my prompt was to make the AI act as a Dungeon Master that could guide the player through character creation and then run a Dungeons & Dragons adventure.

I started the prompt by clearly telling the model that it is a Dungeon Master. This helps the AI stay in character and respond in a consistent way instead of acting like a normal chatbot.

I wrote the prompt so the AI follows a specific order:
Choose a race
Ask for the character’s name
Choose a class
Roll ability scores
Assign skills and equipment

Create a background story
This prevents the AI from skipping steps or doing things out of order.

I included rules like:
Don’t continue without a character name
Always use the character’s name
Never control the player’s actions

I told the model to describe scenes vividly and create a personalized backstory using the character’s race, class, and abilities. This makes the game more fun and engaging.
